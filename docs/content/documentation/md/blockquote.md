## Blockquote 

This theme extend blockquote to highlight text based on the first char.

```md
> This is a standard blockquote
```
>This is a standard blockquote

```md
> ℹ This is an information
```
> ℹ This is an information

```md
> 📎 This is a note
```
> 📎 This is a note

```md
> ❓ This is a question
```
> ❓ This is a question

```md
> ⚠ This is a warning
```
> ⚠ This is a warning 

```md
> 💡 This is a tip
```
